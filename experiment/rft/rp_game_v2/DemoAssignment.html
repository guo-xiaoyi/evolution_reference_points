Developer Mode: Treatment Control

You can see this page because you are in Developer Mode. If you believe you are in a real experiment, please contact the experiment administrator and close this page. When you report, please also include the indication code 0INFO and your Prolific UID.

If you are a researcher, you can use this page to control the treatment assignments for participants (which would be you).

Select the assignment for this participant.

Eligible payoff tree means the payment draw will be selected from the {{ eligible_lottery_count }} tree(s) marked as treatment in the evaluation lottery file. Non-eligible payoff tree means the payment draw will be selected from the {{ ineligible_lottery_count }} remaining evaluation tree(s).

The additional treatment-payment page appears only when both treatment group and eligible payoff tree are selected.
